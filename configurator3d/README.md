# Конфигуратор модульного ЦОД 2D→3D (`configurator3d/`)

Конфигуратор модульного ЦОД с переходом из 2D-компоновки в 3D-визуализацию (рендер на Three.js).

- **Тип:** `ui`
- **Точка входа:** `index.html`
- **Главные файлы:**
  - `index.html` — единый самодостаточный экран: 2D-редактор компоновки, 3D-сцена (three.module), стили и логика inline
- **Расчётная часть (calc):** — (логика встроена в `index.html`)
- **UI/рендер:** `index.html` (THREE / three.module для 3D-сцены)
- **Данные/справочники:** — (нет manifest.json; не зарегистрирован в modules.json)
- **Cross-module связи:** нет
- **Куда добавлять новое:** редактирование сцены и геометрии модулей — в скриптовой части `index.html`. Примечание: модуль не имеет manifest.json и отсутствует в `modules.json` (вероятно, экспериментальный/предшественник `mdc-config`)
