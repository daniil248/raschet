Конфигуратор трансформатора

Подбор силового трансформатора: мощность, первичное / вторичное напряжение, группа соединений, потери ХХ / КЗ, напряжение КЗ uk, тип охлаждения. Результат применим к любому узлу «Источник → Трансформатор» в главной схеме.

⚙ В разработке

Подпрограмма на этапе скелета. Используемая инфраструктура:

  • shared/auth.js — Firebase Auth
  • shared/styles/base.css — дизайн-токены
  • shared/app-header.css / app-header.js

Будущее наполнение: каталог трансформаторов (ABB, Siemens, ЭТК, Тольятти) с XLSX-импортом, каскадный пикер Производитель → Серия → Типоразмер, автоматический расчёт Ik по uk, интеграция с TRANSFORMER_CATALOG главного приложения.
