PDU

Блоки распределения питания. У каждого PDU указывается ввод (A / B / C / D), в который он подключается в электрической схеме. PDU на одном вводе суммируются, PDU на разных вводах резервируют друг друга и их мощность не складывается дважды (аналогично двум вводам потребителя с приоритетом 1). У PDU может быть несколько типов розеток — добавьте строки ниже.

Дополнительные аксессуары

Каталог аксессуаров: полки (лёгкие/тяжёлые), L-направляющие, горизонтальные и вертикальные кабельные органайзеры, анкерные постаменты, фан-боксы, воздухораспределители. Поддерживаются Kehua Wise, APC NetShelter, Rittal TS IT/VX25, Raritan / Minkels. По умолчанию в окне выбора показываются только позиции, соответствующие производителю шкафа — снимите ограничение, если нужна «абстрактная» позиция из другого каталога. Заглушки пустых U настраиваются в отдельном блоке выше.
