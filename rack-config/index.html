Конфигуратор серверной / телеком стойки в разработке

Модуль формирует BOM на заказ стойки: корпус, двери, боковые стенки, пол, кабельные вводы, заглушки на пустые юниты, PDU с привязкой по мощности и количеству розеток нужного стандарта.

Что будет в модуле

  • Корпус: форм-фактор (42U/47U/…), ширина (600/800), глубина (600/800/1000/1200), производитель/серия.
  • Двери: передняя и задняя — стекло / сетка / металл / без двери; замок.
  • Аксессуары: боковые стенки, крыша/верхняя панель, основание/пол, цоколь.
  • Кабельные вводы: кол-во, диаметр, тип (щётка / заглушка / PG).
  • Заглушки: автоматически на все свободные U; тип (1U глухая / 1U перф / 2U).
  • PDU: кол-во × номинал (16/32 А), 1ф/3ф, 0U/1U/2U, набор розеток (C13/C19/Schuko) + кол-во. Проверка соответствия заявленной мощности шкафа.
  • Сохранение шаблонов: одна конфигурация применяется к нескольким идентичным стойкам проекта.

Связанные модули

  • Конфигуратор СКС / телеком — описание содержимого стойки (коммутаторы, патч-панели, серверы) с привязкой к розеткам PDU.
