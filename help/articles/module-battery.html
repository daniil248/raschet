Назначение

Расчёт АКБ — подбор аккумуляторной батареи для систем бесперебойного электропитания (ИБП, СОПТ, АСУ ТП).

3 режима

  • Автономия при заданных блоках — посчитать сколько мин АКБ держит N кВт
  • Сколько блоков для целевого времени — подбор N под Tмин
  • 🤖 Авто-оптимум — программа сама подбирает endV и число блоков (баланс длительность/циклов)

Особенности

  • Импорт XLSX-таблиц разряда производителей (Kehua, Panasonic, SVC, Sonnenschein)
  • Раздельная физика VRLA vs Li-Ion (LFP) — разные коэф. температуры
  • Компоновка VRLA-шкафа
