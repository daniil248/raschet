Что это

Центр помощи — справочник по платформе Raschet. Открывается из Hub'а (главного экрана) или прямой ссылкой /help/. Состоит из набора статей в 4 категориях.

Структура

Категории статей

КатегорияСодержимое
🏁 Начало работыВведение в платформу, обзор проектов и модулей
📦 МодулиОписание каждого модуля (Конструктор / Технолог / СКС / МЦОД и др.)
✨ ВозможностиКросс-модульные фичи: пресеты, статусы, режимы хранения и т.д.
🏗 АрхитектураВнутреннее устройство: data-model, ключи storage

Поиск

В шапке sidebar — поле поиска. Ищет по названиям и summary статей. Категории, в которых нет совпадений, скрываются. Очистка поля — возвращает полный список.

Cross-references

Внутри статей есть ссылки вида «вот эта» (с пунктирной линией) — клик переключает на связанную статью с обновлением breadcrumb и блока «См. также» внизу.

Список «См. также» (related[] в манифесте) под каждой статьёй автоматически фильтруется по доступу к модулям.

Доступ к статьям

Каждая статья может иметь поле requiresModule. Если этот модуль отключён в modules.json (поле enabled: false) — статья:

  • Скрыта в sidebar
  • Скрыта в «См. также»
  • При открытии прямой ссылкой (#article-id) показывает «Доступ ограничен»

Это аналог permission'ов модулей — пользователь видит ровно то, что соответствует его конфигурации платформы.

Прямые ссылки

URL вида /help/#feature-card-presets открывает конкретную статью. Hash-routing — можно делиться ссылкой с коллегой, она откроет ту же статью.

Как добавлять статьи

  1. Создайте HTML-файл в help/articles/<id>.html с содержимым (без <html> / <body> — только тело).
  2. Добавьте запись в help/articles.json с id, title, category, summary, file, related[], опционально requiresModule.
  3. Используйте классы .note / .warn / .ok для блоков предупреждений.
  4. Inline SVG для диаграмм с классом hc-fig.
  5. Cross-ref: <a data-article-link href="#other-id">…</a>
