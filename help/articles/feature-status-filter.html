Жизненный цикл проекта

Каждый проект имеет статус, отражающий стадию жизненного цикла. По умолчанию новый проект — draft (черновик).

Multi-status фильтр

В списке проектов (модуль 📁 Проекты) над сеткой карточек — chip-bar с группировкой:

Каждый chip — toggle:

  • Активный (цветной фон) — проекты этого статуса видимы в списке
  • Неактивный (серый) — скрыты
  • Цифра справа — количество проектов в этом статусе

«Все» — кнопка для быстрого включения всех статусов.

Сохранение фильтра

Текущий состав активных статусов сохраняется в LocalStorage (raschet.projects.statusFilter.v1) — между сеансами. По умолчанию (при первом входе) — все статусы кроме «Архив».

Как сменить статус проекта

В карточке проекта на главной странице (📁 Проекты) кнопка с текущим статусом — клик открывает picker всех 5 статусов. Также доступно из карточки конкретного проекта (project.html) — кнопка Статус ▾.

💡 Совет: для большой команды — фильтр «Проектируется» + «Смонтирован» даст актуальный список проектов в работе, без черновиков и архива.
