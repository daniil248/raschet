Назначение

Конфигуратор ИБП — подбор конфигурации ИБП по нагрузке: тип (моноблок / модульный), резервирование N+X, параметры входа и батарейной цепи.

Связи

  • Конструктор схем — UPS-узлы привязаны к моделям из ИБП-каталога
  • Расчёт АКБ — подбор батарей под выбранный ИБП
  • Технолог ЦОД — концептуальная стадия (purpose: IT / cooling)
