Зачем

Карточки узлов на схеме (consumer / panel / source / generator / ups) могут содержать много полей: kW, A, автомат, кабель, ΔU, cosφ, фаза, габариты и т.д. Не всем пользователям нужен весь набор:

  • Электрик хочет видеть автомат, Iдоп, кабель, ΔU.
  • Технолог — мощность, габариты, тип охлаждения.
  • Презейл / клиент — только tag/name (для скриншотов).

Где переключается

В шапке инструментов (toolbar) Конструктора схем — селектор 🎴 ▾. Слева от него — иконка карточки. Справа — кнопка ✎ (открыть редактор пресетов).

Иерархия выбора

На текущий момент UI редактора поддерживает только User-уровень. Project и Scheme — поддержаны в resolver (можно задать через JSON), UI для них в разработке.

Редактор пресетов (v2)

Редактор открывается кнопкой ✎ рядом с селектором. Возможности (v0.59.799+):

  • Перемещаемая модалка — drag за заголовок (на curвсор «move»)
  • Двухколоночный layout: слева — список полей с чекбоксами + drag-источник, справа — live-preview карточки с зонами
  • 4 зоны на карточке: заголовок (вверху, для tag/name), иконка (вверху-справа), тело (центр) и подвал (внизу)
  • Drag-drop полей: перетащите чип из любой зоны в другую, или поле из левого списка прямо в зону
  • Editable labels зон — кликните на текст «Заголовок / Иконка / Тело / Подвал» и впишите свой
  • Чип × — убрать поле из зоны (поле выключается из пресета)
  • За один раз — одна карточка: выбран type-tab (consumer / panel / source / generator / ups / zone / channel) — настраивается её preview
  • Per-mode + per-type: разные настройки для schematic / layout / scs-design + каждого типа узла

Custom user-пресеты

  1. Click на ✎ рядом с селектором → открывается редактор.
  2. Tab «📋 Пресеты»: системные read-only вверху, user внизу.
  3. Кнопка ➕ Создать — копирует выбранный пресет (или Полный).
  4. Tab «✎ Поля»: per-mode (schematic / layout / scs-design) → per-type (consumer / panel / ...) → чекбоксы по полям.
  5. Required-поля (tag/name) заблокированы — всегда on.
  6. Кнопки ☑ Все / ☐ Только обязательные — bulk-операции.

Live-переключение

Смена активного пресета через селектор → raschet:card-preset-changed event → invalidate cache в render.js → re-render всех карточек canvas. Без перезагрузки страницы.

Импорт / экспорт

Tab «⇅ Импорт/Экспорт» в редакторе:

  • ⬇ Скачать JSON — экспорт всех user-пресетов в файл card-presets-YYYY-MM-DD.json.
  • ⬆ Загрузить JSON — импорт. Существующие пресеты с тем же id обновляются.

Реестр полей

Полный список полей и их группировка — в shared/card-fields-registry.js:

  • identification: tag*, name*, count, icon, sourceSubtype, ...
  • electrical: demandKw, currentA, breakerIn, cableSpec, deltaUPct, voltage, phase, cosPhi, capacityA, marginPct, kVA, ...
  • mechanical: widthMm, depthMm, heightMm, weightKg, rackUnits, cooling
  • status: backupMode, redundancy, switchMode, sectionsCount

* — обязательные (required, нельзя выключить).
