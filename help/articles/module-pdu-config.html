Назначение

Конфигуратор PDU — Power Distribution Unit для серверной стойки.

Параметры

  • Тип (Basic / Metered / Switched / Monitored)
  • Фазность (1ф / 3ф)
  • Номинальный ток (16 / 32 / 63 А)
  • Розетки (C13/C19, число)
  • Резервирование (1 / 2 / 4 ввода на стойку)

Из Технолог ЦОД PDU настраивается per-rack-group (см. статью).
