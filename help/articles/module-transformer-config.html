Назначение

Конфигуратор трансформатора — справочник силовых трансформаторов (ТМГ, ТСЛ, ТМЗ, ТСЗГЛ, ТМ, ТМН) с расчётом IКЗ по паспортным данным.

Параметры

  • Поставщик / серия / S (кВА)
  • Первичное / вторичное напряжение, группа соединений (Dyn11, Yyn0, ...)
  • uk (%), потери ХХ / КЗ
  • Тип охлаждения (ONAN, ONAF, AN, AF)

Доступен из инспектора источника Конструктора через transformer-picker.
