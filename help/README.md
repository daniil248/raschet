# Центр помощи (`help/`)

Справочник по платформе: модули, возможности, архитектура. Статьи с поиском, cross-references и SVG-диаграммами. Доступ зависит от подключённых модулей.

- **Тип:** `ui`
- **Точка входа:** `index.html`
- **Главные файлы:**
  - `help.js` — экран справки: навигация, поиск, рендер статей
  - `articles.json` — индекс статей (метаданные, связи, доступ по модулям)
  - `articles/*.html` — контент статей (intro, module-*, feature-*, arch-*)
- **Расчётная часть (calc):** —
- **UI/рендер:** `help.js` + `help.css`
- **Данные/справочники:** `articles.json` + каталог `articles/`
- **Cross-module связи:** нет (видимость статей зависит от подключённых модулей)
- **Куда добавлять новое:** новая статья — HTML-файл в `articles/` + запись в `articles.json`; логика навигации/поиска — в `help.js`
