Конфигуратор ИБП

Подбор конфигурации ИБП по нагрузке: тип (моноблок / модульный), резервирование N+X, параметры входа и батарейной цепи. Результат можно применить к любому узлу ИБП в главной схеме.

⚙ В разработке

Подпрограмма на этапе скелета. Инфраструктура уже работает:

  • shared/auth.js — Firebase Auth (инициализирован)
  • shared/styles/base.css — дизайн-токены и кнопки
  • shared/app-header.css / app-header.js — единый хедер
  • shared/ups-picker.js — API для чтения параметров ИБП с узла

Будущее наполнение: каскадный пикер моделей ИБП (Производитель → Серия → Модель) + применение к узлу, расчёт резервирования, предпросмотр структурной схемы.
