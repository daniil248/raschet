🏭 Реестр оборудования объекта Планируется

Единый журнал всего не-IT-имущества площадки: ИБП, АКБ, кондиционеры, ДГУ, трансформаторы, щиты, СКУД, системы пожаротушения. Экземпляры с привязкой к помещению / модулю ЦОД / линии питания. Общая крыша над «Реестром IT-оборудования» на уровне «всё имущество объекта». Roadmap: Подфаза 1.26 (п. 1.26.5).

Что будет внутри

  • Экземпляры — каждый физический предмет (не тип): инв-№, серийный, заводской, место установки, дата ввода в эксплуатацию, гарантия, межрегламентный пробег.
  • Группировка по классам — силовое, слаботочное, инженерное, климат, безопасность.
  • Привязка к проекту/площадке — здание, этаж, помещение, стойка (для стоечных единиц) или модуль ЦОД.
  • Импорт из модулей-конфигураторов — при сохранении конфигурации в mv-config / ups-config / mdc-config / suppression-config экземпляры попадают сюда автоматически.
  • Жизненный цикл — установлено / на складе / в ремонте / списано. История перемещений.
