Конфигуратор НКУ (LV, до 1 кВ)

Низковольтные комплектные устройства по IEC 61439 (TTA / PTTA): ВРУ, ВРУ с АВР, ЩС, ЩО, ЩК. Для РУ среднего напряжения (6-35 кВ) — отдельный Конфигуратор РУ СН.

🧙 Конфигуратор щита для проекта

Шаг 1. Требования из проекта

Параметры переданы из Конструктора схем. Можно подправить.

Шаг 2. Подбор оболочки (корпуса)

Оболочки из справочника, подходящие по току, IP и форме.

Шаг 3. Состав автоматов

Вводной автомат + N автоматов отходящих линий. Детальный подбор (настройки, селективность) — в Фазе 1.10 (справочник автоматов).

Шаг 4. Учёт, ТТ, мониторинг, аксессуары

Измерительные трансформаторы, приборы учёта, мониторинг и навесное оборудование. Можно включать для всех линий, только вводов или выборочно.

Класс точности 0.5S обязателен для расчёта с сетевой компанией. Требует ТТ класса 0.5S (включается автоматически ниже).
Авто-номинал: ближайший стандартный ряд ≥ 1.25 · Iₙ автомата (ГОСТ 7746 / IEC 61869-2).
Принадлежности из каталога
Реле напряжения, лампы сигнализации, замки, обогрев, вентиляторы, термостаты, шинодержатели и т.п. Позиции войдут в спецификацию.
