# Методики газового пожаротушения (`suppression-methods/`)

Переиспользуемая библиотека методик расчёта газового пожаротушения (без UI). Реестр методик с единым интерфейсом: `{ META, compute(input) }`, вход `{ agent, V, fireClass, leakage, tempC, altM, designFactor? }` → `{ M, C, steps[], ... }`.

- **Тип:** `calc-lib`
- **Точка входа:** `index.js` (registry методик)
- **Главные файлы:**
  - `index.js` — реестр методик с унифицированным интерфейсом
  - `sp-rk-2022.js`, `sp-485-2020.js`, `sp-485-annex-d.js`, `nfpa-2001.js`, `iso-14520.js` — нормативные методики
  - `agents.js` — справочник ГОТВ; `modules-catalog.js` — каталог модулей/баллонов
  - `hydraulics.js` — гидравлический расчёт; `report-text.js` — текст для отчёта
- **Расчётная часть (calc):** все файлы — чистая логика без DOM
- **UI/рендер:** — (нет index.html; чистая библиотека)
- **Данные/справочники:** `agents.js`, `modules-catalog.js`; нормативные коэффициенты в файлах методик
- **Cross-module связи:** потребляется модулем `suppression-config/` (импорт `../suppression-methods/*`)
- **Куда добавлять новое:** новая методика/норматив — отдельный файл + регистрация в `index.js`; тесты — в `validation-tests.js`
