📦 Каталог типов оборудования

Базовые типы СКС-оборудования. Хранятся локально (раздел каталога «пользовательские типы»). Из каталога единицы добавляются в стойку через кнопку ➕.

🔌 Размещение в стойке

Каждая единица занимает N U сверху вниз. Позиция U — верхний юнит устройства. Красная метка — конфликт (наезд на другое / на заглушки).

🧵 СКС-матрица (патч-корды)

Простой список соединений «порт A ↔ порт B» для BOM патч-кордов.

🗺️ Карта юнитов

🛒

Тележка 0

Межстоечный буфер проекта. Перетащите устройство из карты юнитов на тележку, чтобы вытащить. Перетащите строку с тележки на карту любой стойки — установить. Видна при открытии любой стойки.

📦

Склад 0

Долговременное хранение оборудования проекта. С тележки «→ на склад»; со склада «→ на тележку» (и дальше — в любую стойку).
