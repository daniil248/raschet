# `elements/` — постоянная редирект-заглушка

Исторический модуль «Библиотека элементов» **объединён** с модулем
«Каталог и библиотека» (`apps/catalog/`). Папка сохранена ТОЛЬКО как
постоянный редирект для старых закладок/ссылок на `/raschet/elements/`.

- **Тип:** редирект-заглушка (НЕ модуль; нет manifest, нет записи в
  `modules.json`, нет карточки в `/modules/`).
- **`index.html`** — страница с `meta-refresh` на `../catalog/` (2 с) +
  пояснение «модуль объединён». Удалять НЕ нужно: ловит старые URL.
- **`elements.css`** — минимальные стили самой заглушки (используется
  `index.html`).
- **`elements-editor.js`** — УДАЛЁН (v0.60.571): устаревший MVP-CRUD,
  0 живых импортёров, дублировал `apps/catalog/`. История в git.
- **Куда новое:** только в `apps/catalog/` (там редактор элементов +
  цены/контрагенты/импорт/аналитика над тем же `shared/element-library`).
