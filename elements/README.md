# Библиотека элементов — переезд (`elements/`)

Исторический модуль управления библиотекой элементов. Объединён с модулем «Каталог и библиотека» (`catalog/`); страница автоматически перенаправляет в `../catalog/`.

- **Тип:** `ui`
- **Точка входа:** `index.html` (meta-refresh-редирект на `../catalog/` через 2 с)
- **Главные файлы:**
  - `index.html` — страница-заглушка с редиректом на `catalog/`
  - `elements-editor.js` — устаревший минимальный CRUD-редактор `shared/element-library` (Фаза 1.1.3 MVP)
  - `elements.css` — стили
- **Расчётная часть (calc):** —
- **UI/рендер:** `elements-editor.js` (legacy)
- **Данные/справочники:** `shared/element-library` (актуальная работа — в модуле `catalog/`)
- **Cross-module связи:** редирект на `catalog/`
- **Куда добавлять новое:** ничего не добавлять — функциональность перенесена в `catalog/`; новое — туда
