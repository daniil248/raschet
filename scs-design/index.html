📡 Проектирование СКС Планируется

Модуль для проектирования структурированной кабельной системы объекта: трассы, кроссы, пары порт ↔ порт на уровне здания и этажей, маркировка, длины кабелей, BOM СКС. Roadmap: Подфаза 1.26 / бывшая 1.25.

Что будет внутри

  • План зала — top-down план с расстановкой стоек (импорт координат из mdc-config или ручное размещение).
  • Стойки проекта — список из Шкафы проекта с их external-портами (1.24.15).
  • Связи port ↔ port — соединения между внешними портами стоек с типом кабеля (copper / OM3 / OS2), длиной и маркировкой.
  • BOM СКС — агрегированный по типу кабеля, с коэффициентом запаса длины.
  • Экспорт — план в SVG/PDF, кабельный журнал в XLSX.

🧰 Мастер меж-шкафных связей

Встроенный рабочий стол внутри этого модуля: выбираете несколько стоек проекта — они открываются рядом, можно тянуть линию от конкретного порта одного шкафа к порту или патч-панели другого. Отдельного пункта в хабе нет — мастер доступен только отсюда, вкладка «Связи».

Что уже можно сейчас

Внутренние патчкорды внутри одного шкафа — в Компоновщике шкафа (СКС-матрица). Этот модуль поднимается на уровень выше — трассы между шкафами и кабельный журнал всего объекта.
