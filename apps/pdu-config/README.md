# Конфигуратор PDU (`pdu-config/`)

Standalone-подбор PDU из единой библиотеки элементов: расчёт тока стойки по контексту (серверов, кВт/серв, cos φ, 2N), автоподбор номинала и количества розеток C13, ранжирование вариантов score 0–100.

- **Тип:** `ui`
- **Точка входа:** `index.html`
- **Главные файлы:**
  - `pdu-config.js` — расчёт тока стойки, автоподбор PDU, ранжирование по покрытию/номиналу/перерасходу
  - `pdu-config.css` — стили
- **Расчётная часть (calc):** логика подбора/ранжирования внутри `pdu-config.js` (отдельный calc-слой не выделен)
- **UI/рендер:** `pdu-config.js`
- **Данные/справочники:** `listElements({kind:'pdu'})` из `shared/element-library` (через catalog-bridge → `shared/catalogs/pdus.js`); (нет manifest.json; не зарегистрирован в modules.json)
- **Cross-module связи:** контекст стойки на входе (серверы, кВт, cos φ, 2N)
- **Куда добавлять новое:** модели PDU — через `shared/element-library`; алгоритм score и экран — в `pdu-config.js`
