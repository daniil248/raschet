# Расчёт кабельной линии (`cable/`)

Подбор сечения кабеля по IEC / ПУЭ с учётом температуры, группирования, способа прокладки, экономической плотности тока и падения напряжения.

- **Тип:** `ui`
- **Точка входа:** `index.html`
- **Главные файлы:**
  - `cable-calc.js` — UI-оркестратор и расчёт сечения (IEC/ПУЭ, поправки, падение напряжения)
  - `protection-wizard.js` — мастер подбора аппарата защиты под линию
  - `cable-calc.css` — стили экрана
- **Расчётная часть (calc):** логика расчёта внутри `cable-calc.js` (отдельный calc-слой не выделен)
- **UI/рендер:** `cable-calc.js`, `protection-wizard.js`
- **Данные/справочники:** глобальные настройки `raschet.global.v1`; `shared/calc-modules`
- **Cross-module связи:** `shared/report` для экспорта; URL-параметр `project`
- **Куда добавлять новое:** новые поправочные коэффициенты/нормы — в `cable-calc.js`; логику подбора защиты — в `protection-wizard.js`
