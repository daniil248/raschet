# Конфигуратор модульного ЦОД (`mdc-config/`)

Конфигуратор модульного ЦОД серии GDM-600: цепочка типовых модулей, стены/рама/фальшпол, тамбур, ODU на балконе, климат, cosφ, загрузка ИБП, Li-ion АКБ, трубопровод АГПТ.

- **Тип:** `ui`
- **Точка входа:** `index.html`
- **Главные файлы:**
  - `mdc-config.js` — конфигуратор: модель данных (последовательность модулей), геометрия, 3D-вид
  - `mdc-config.css` — стили
- **Расчётная часть (calc):** логика встроена в `mdc-config.js` (отдельный calc-слой не выделен)
- **UI/рендер:** `mdc-config.js`
- **Данные/справочники:** типовые модули — из каталога; (нет manifest.json; не зарегистрирован в modules.json)
- **Cross-module связи:** климат/cosφ/загрузка ИБП — параметры конфигурации
- **Куда добавлять новое:** новые типы модулей/геометрию — в `mdc-config.js`; при стабилизации добавить manifest.json и запись в `modules.json`
