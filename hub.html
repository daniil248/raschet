Скрытые модули (клик — вернуть):

Конструктор схем

Проектирование принципиальных схем электроснабжения с автоматическим расчётом нагрузок, подбором кабелей и защиты

Активен

Расчёт кабельной линии

Подбор сечения кабеля по IEC / ПУЭ с учётом температуры, группирования, способа прокладки, экономической плотности тока и падения напряжения

Новый

Схема принципиальная

Редактор принципиальных электрических схем и рабочей документации. Библиотека IEC 60617-DB-12M, листы ISO 216 (A4…A0), основная надпись по ISO 7200. Используется Конструктором схем для формирования щитов

Новый

Расчёт АКБ

Подбор ёмкости и конфигурации АКБ для ИБП. Загрузка таблиц разряда производителей (Kehua, Panasonic, SVC, Sonnenschein) в XLSX и точный расчёт по ним. Если данных нет — усреднённая модель по химии батареи

Новый

Конфигуратор ИБП

Подбор конфигурации ИБП по нагрузке: тип (моноблок / модульный), резервирование N+X, параметры входа и батарейной цепи. Результат можно применить к любому узлу ИБП в главной схеме

Конфигуратор НКУ (LV щит)

Низковольтные комплектные устройства до 1 кВ (IEC 61439): ВРУ, АВР, ЩС, ЩО, ЩК. Подбор оболочки, вводные/отходящие автоматы, секционирование, формы разделения. Для MV — отдельный «Конфигуратор РУ СН»

Новый

Шаблоны отчётов

Подготовка и хранение шаблонов оформления отчётов: страница, поля, логотип, колонтитулы, стили заголовков. Используются подпрограммами для экспорта в PDF и DOCX

Новый

Каталог и библиотека элементов

Единый справочник оборудования платформы: ИБП, щиты, АКБ, трансформаторы, автоматы, корпуса, климат, кабели + цены по датам и контрагентам + импорт прайс-листов XLSX + аналитика. Один источник данных — никакого дублирования

Новый

Рабочее место логиста

Расчёт доставки, маршруты, склады, тарифы перевозчиков. Интеграция с BOM проекта: оборудование + логистика = полная смета с учётом доставки до объекта

В разработке

Конфигуратор РУ СН

Распределительные устройства 6-35 кВ: моноблоки SF6 (RM6, SafeRing) и сборные (ЩО-70). Справочник + wizard-подбор с конфигурированием ячеек (ввод, отходящая, защита ТП, измерения, секционная)

Новый

Roadmap

План архитектурного развития платформы: выполненные фазы, текущие задачи, очередь и долгосрочные цели. Источник — ROADMAP.md в репозитории

Актуально

История изменений

Журнал версий платформы: что добавлено, исправлено и изменено в каждом релизе. Поиск по версиям и описанию. Ссылки на коммиты в GitHub

Актуально

Конфигуратор трансформатора

Подбор силового трансформатора: мощность, первичное / вторичное напряжение, группа соединений, потери ХХ / КЗ, uk, тип охлаждения. Результат применим к узлу «Источник → Трансформатор»

В разработке

ID-диаграмма Молье-Рамзина

Психрометрический расчёт влажного воздуха по ASHRAE Fundamentals 2021 (Hyland-Wexler 1983). Состояние: t, φ, d (W), h, tр, tм, ρ, v. HVAC-процессы: нагрев, охлаждение с осушением (bypass factor), адиабатическое и паровое увлажнение, смешение с рециркуляцией. SVG-диаграмма ID с изолиниями φ, h.

В разработке

Расчёт газового пожаротушения (АГПТ)

Расчёт массы ГОТВ, подбор баллонов, диаметр трубопровода, количество форсунок. Подключаемые методики: СП РК 2.02-102-2022, СП 485.1311500.2020, NFPA 2001, ISO 14520. Каталог ГОТВ: FM-200, Novec 1230, HFC-125, CO₂, IG-541/55/100/01, SF₆. Отчёт XLSX/HTML, изометрия трубопровода, спецификация.

В разработке

Конфигуратор модульного ЦОД

Модульный ЦОД серии GDM-600: ширина 2400/3000 мм, длина ISO 20ft/40ft, сборка нескольких модулей. Опросник: IT-мощность, резервирование, ИБП+АКБ, охлаждение (inRow/perimeter/freecooling), слаботочка. Расстановка стоек, ИБП+АКБ, CRAC в отдельные зоны. Top-view планировка, BOM, выгрузка объёма поставки

В разработке

Конфигуратор шкафа — корпус

Типовое железо шкафа: корпус (U, ширина, глубина), двери (стекло / сетка / металл), боковые стенки, пол, кабельные вводы, заглушки пустых U, PDU-слоты (C13/C19/Schuko, 1ф/3ф, 16/32 А), заземление. Шаблоны и BOM пустого шкафа. Наполнение конкретного экземпляра — в «Компоновщике шкафа»

Конфигуратор PDU

Подбор PDU по требованиям: число фаз, номинал (А), категория (basic / metered / monitored / switched / hybrid), состав розеток (C13/C19/Schuko), высота (0U/1U/2U), производитель. Результат — список подходящих артикулов от APC, Rittal, Raritan, Kehua с возможностью экспорта спецификации

Компоновщик шкафа

Наполнение конкретного экземпляра шкафа: карта юнитов (коммутаторы, патч-панели, серверы 1U/2U/4U, КВМ, мониторы), вертикальные PDU, внутренние патчкорды и питание. Привязка устройств к розеткам PDU и матрица внутренних соединений. Корпус — из «Конфигуратора шкафа — корпус»

Активный

Реестр IT-оборудования

Парк IT-экземпляров проекта: сервера, коммутаторы, хранилища с inv-№, S/N, IP, MAC, физическими портами. Привязка к шкафу и U. Поиск, фильтры по месту (в шкафу / тележка / склад), импорт и экспорт

Новый

Проектирование СКС

Структурированная кабельная система объекта: мастер меж-шкафных связей, план зала с расстановкой стоек (drag-n-drop), автоматические длины кабелей по манхэттен-расстоянию, кабельный журнал (BOM) с агрегацией по типам, CSV-экспорт. Патчкорды внутри одного шкафа — в «Компоновщике шкафа»

Активный

Реестр оборудования объекта

Всё не-IT имущество площадки: ИБП, АКБ, кондиционеры, ДГУ, ТП, щиты, СКУД, пожарные системы. Экземпляры с привязкой к помещению и модулю ЦОД. Общая крыша над реестром IT на уровне «всё имущество площадки»
