Конструктор схем

Проектирование принципиальных схем электроснабжения с автоматическим расчётом нагрузок, подбором кабелей и защиты

Активен

Расчёт кабельной линии

Подбор сечения кабеля по IEC / ПУЭ с учётом температуры, группирования, способа прокладки, экономической плотности тока и падения напряжения

Новый

Схема принципиальная

Редактор принципиальных электрических схем и рабочей документации. Библиотека IEC 60617-DB-12M, листы ISO 216 (A4…A0), основная надпись по ISO 7200. Используется Конструктором схем для формирования щитов

Новый

Расчёт АКБ

Подбор ёмкости и конфигурации АКБ для ИБП. Загрузка таблиц разряда производителей (Kehua, Panasonic, SVC, Sonnenschein) в XLSX и точный расчёт по ним. Если данных нет — усреднённая модель по химии батареи

Новый

Конфигуратор ИБП

Подбор конфигурации ИБП по нагрузке: тип (моноблок / модульный), резервирование N+X, параметры входа и батарейной цепи. Результат можно применить к любому узлу ИБП в главной схеме

В разработке

Конфигуратор щита

Подбор состава распределительного щита: ввод / АВР, секционирование, номиналы вводных и отходящих автоматов, шинопровод. Результат применим к узлу «Щит» в главной схеме

В разработке

Шаблоны отчётов

Подготовка и хранение шаблонов оформления отчётов: страница, поля, логотип, колонтитулы, стили заголовков. Используются подпрограммами для экспорта в PDF и DOCX

Новый

Каталог и библиотека элементов

Единый справочник оборудования платформы: ИБП, щиты, АКБ, трансформаторы, автоматы, корпуса, климат, кабели + цены по датам и контрагентам + импорт прайс-листов XLSX + аналитика. Один источник данных — никакого дублирования

Новый

Рабочее место логиста

Расчёт доставки, маршруты, склады, тарифы перевозчиков. Интеграция с BOM проекта: оборудование + логистика = полная смета с учётом доставки до объекта

В разработке

Конфигуратор РУ СН

Распределительные устройства 6-35 кВ: моноблоки SF6 (RM6, FafeRing) и сборные (ЩО-70). Справочник + wizard-подбор с конфигурированием ячеек (ввод, отходящая, защита ТП, измерения, секционная)

Новый

Конфигуратор трансформатора

Подбор силового трансформатора: мощность, первичное / вторичное напряжение, группа соединений, потери ХХ / КЗ, uk, тип охлаждения. Результат применим к узлу «Источник → Трансформатор»
